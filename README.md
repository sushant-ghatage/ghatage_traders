# Ghatage Traders - Business Website

A beautiful, modern static website for Ghatage Traders - Wholesale and Retail Dealers.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with purple and beige color scheme
- **Multi-Page Navigation**: Easy navigation between all pages
- **Product Categories**: Showcases four main product categories:
  - Household Plastic Items
  - Puja Sahitya (Religious Items)
  - Kitchen Appliances
  - Atta Chakki (Flour Mills)
- **Contact Form**: Interactive contact form for customer inquiries
- **Mobile Menu**: Hamburger menu for mobile devices

## Pages

1. **index.html** - Home page with product category overview
2. **products.html** - Detailed product listings for all categories
3. **about.html** - Information about the business, mission, and values
4. **contact.html** - Contact information and inquiry form

## How to Use

Simply open `index.html` in any modern web browser. No server or installation required!

### Option 1: Direct Open
Double-click `index.html` to open it in your default browser.

### Option 2: Local Server (Recommended)
For the best experience, you can run a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then open `http://localhost:8000` in your browser.

You can easily customize:
